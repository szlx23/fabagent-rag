## 收集建议

- 优先找公开、可下载、没有版权或保密风险的资料。
- 每个文件下载后建议记录来源 URL，方便后续复查解析结果。
- 文件名建议包含类型和用途，例如 `pdf_sop_etch_001.pdf`。
- 扫描件、截图、手机拍摄文档要单独保留，用来验证 OCR 效果。
- 每类文件至少准备 1~2 个“质量较差”的样本，例如低清晰度、表格复杂、标题层级混乱。
